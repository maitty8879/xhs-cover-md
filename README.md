# 🎨 XHS Cover from DESIGN.md

> Turn any company's DESIGN.md into stunning Xiaohongshu (小红书) covers with AI.

把 58 家大厂的 [DESIGN.md](https://github.com/VoltAgent/awesome-design-md) 设计规范，一键转化为小红书 3:4 封面。

![covers](https://img.shields.io/badge/covers-58_brands-ff2442) ![format](https://img.shields.io/badge/format-1500×2000px-blue) ![license](https://img.shields.io/badge/license-MIT-green)

## How it works

```
DESIGN.md (大厂设计规范)
    ↓  Extract visual DNA (color, typography, components)
    ↓  Generate HTML cover (1500×2000)
    ↓  Screenshot → PNG
    → Post to Xiaohongshu 🚀
```

## Quick Start

### 1. Pick a style
Browse the `covers/` directory — each folder has a `preview.png` and `template.html`.

### 2. Customize
Open `template.html`, change the headline, subtitle, and feature cards to your content.

### 3. Screenshot
```bash
# Using the included script
chmod +x scripts/capture.sh
./scripts/capture.sh covers/apple/template.html output.png

# Or open in Chrome at 1500×2000 viewport and screenshot
```

### 4. Feed DESIGN.md to AI for new styles
```
Generate a 1500×2000px XHS cover HTML in {Brand}'s design language.
Use colors, typography, and components from this DESIGN.md:

[Paste DESIGN.md content]
```

```
│     [CTA]  [CTA]               │
│                                 │
├─────────────────────────────────┤
│  ┌─────┐ ┌─────┐ ┌─────┐      │
│  │  1  │ │  2  │ │  3  │      │  ← Feature cards
│  └─────┘ └─────┘ └─────┘      │
├─────────────────────────────────┤
│  (Tag) (Tag) (Tag) (Tag)        │  ← Brand nav
└─────────────────────────────────┘
```
